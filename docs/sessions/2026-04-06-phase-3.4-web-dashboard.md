## Implementation Details

### 페이지 구조
```
/                   메인 페이지 (리포트 목록, 검색)
/reports/{id}       리포트 상세 페이지
```

### REST API 엔드포인트
```
GET /api/reports         리포트 목록 (JSON)
GET /api/reports/{id}    리포트 상세 (JSON)
GET /api/search          기사 검색 (JSON)
GET /api/categories      카테고리 목록 (JSON)
GET /health              헬스 체크
```

### CLI 옵션
```
--serve           웹 대시보드 서버 시작
--port PORT       포트 지정 (기본: 8000)
```
