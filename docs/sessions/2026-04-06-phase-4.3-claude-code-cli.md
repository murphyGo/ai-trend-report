## Implementation Details

### Claude Code CLI 모드
```bash
claude --bare --model "$MODEL" -p "프롬프트" --allowedTools "Bash,Read,Write,Edit"
```

- `--bare`: OAuth 인증 스킵, ANTHROPIC_API_KEY 환경 변수 사용
- `--model`: sonnet (기본), haiku (저렴), opus (고성능)
- `-p`: Print 모드 (non-interactive)
- `--allowedTools`: 허용할 도구 목록

### 워크플로우 입력 옵션
```yaml
inputs:
  dry_run: boolean     # Slack 전송 없이 테스트
  limit: string        # 기사 수 제한
  use_api: boolean     # API 모드 사용 (기본: false = CLI 모드)
  model: choice        # 모델 선택 (sonnet/haiku/opus)
```
