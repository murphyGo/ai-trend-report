## Key Decisions

| Decision | Rationale |
|----------|-----------|
| ThreadPoolExecutor 선택 | aiohttp 대신 선택 - 기존 코드 변경 최소화, 의존성 추가 없음 |

| URL 기반 캐시 | 기사 내용이 아닌 URL로 중복 판단 (단순하고 빠름) |

```python
def collect_articles_parallel(collectors: list, max_workers: int = 3) -> list:
    with ThreadPoolExecutor(max_workers=max_workers) as executor:
        future_to_collector = {
            executor.submit(collector.collect): collector
            for collector in collectors
        }
        for future in as_completed(future_to_collector):
            # process results
```
