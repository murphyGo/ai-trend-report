## Key Decisions
| Decision | Rationale |
|----------|-----------|
| 파일 이동 안 함 (notifiers/ 패키지화 X) | import 경로 변경은 main.py + 전 테스트에 영향 → 최소 churn |
| `BaseNotifier.send_error_notification` default no-op | Discord는 이 기능 미구현. default False + 경고 로그로 안전하게 처리 |
| `CATEGORY_ORDER`를 constants에 배치 | Slack + Email이 동일 순서로 카테고리 렌더 → 한 곳에서 관리 |
| `is_quiet_day` static method | report만 필요하고 인스턴스 상태 불요 |
