## Implementation Details

### EmailConfig
```python
@dataclass
class EmailConfig:
    enabled: bool = False
    smtp_host: str = "smtp.gmail.com"
    smtp_port: int = 587
    use_tls: bool = True
    username: str = ""
    password: str = ""
    sender: str = ""
    recipients: list[str] = field(default_factory=lambda: [
        "chk8072@naver.com",
        "murphy9333@gmail.com"
    ])
```

## Usage Examples

```bash

# 특정 수신자에게 전송

# 여러 수신자에게 전송
```
